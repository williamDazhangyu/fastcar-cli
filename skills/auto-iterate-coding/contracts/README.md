# Contracts

这里放机器可检查的强约束。

## 作用

- 固定目录职责和文件边界。
- 固定状态、交付和命名契约。
- 让第三方项目知道哪些内容不能随意改写。

## 建议内容

- session 状态契约。
- 交付门禁契约。
- 目录结构契约。
- 版本兼容契约。
